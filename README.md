
# **DailyQuiz App**

 **Daily Quiz** — приложение для прохождения викторин.

### Что удалось реализовать:

* Стартовый экран:
  *  переходы к  викторинам и истории;
  *  отображение лоадера при загрузке вопросв;
  *  сообщение об ошибке при запросе;
